Some extra notes

  • Slicing array using v[start:end:step]. It can be used with python list, tuple or numpy.ndarray (vectors created by calling np.arrray([1,21,3]) for example). Read more at NumPy Array Slicing. 
import numpy as np
v = np.array([1, 2, 3, 4, 5])

type(v)
>>> numpy.ndarray

v[0::] # select all indices, defaut step 1
>>> array([1, 2, 3, 4, 5])

v[0:-1:] # select all indices except the last one
>>> array([1, 2, 3, 4])

v[0:3:] # select only indices 0, 1, 2
>>> array([1, 2, 3])

v[0:3:2] # select only indices 0, 2 as the step is 2
>>> array([1, 3])
  • Avoid using the comparison 1==1.0 or generally a==b when comparing two scalars. Use np.isclose() instead. 
import numpy as np
np.isclose(1, 1.0)
>>> True
  • Useful numpy commands:
    • np.ones create a vectors with all element as 1
    • np.zeros create a vectors with all element as 0 
np.ones(4)
>>> array([1., 1., 1., 1.])

np.zeros(4)
>>> array([0., 0., 0., 0.])
