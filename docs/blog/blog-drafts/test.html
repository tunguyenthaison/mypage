Some notes on Hele-Shaw equation

Some naive readings on the Hele-Shaw equation

If the domain $\Omega$ is given by $\Omega = {(x,y)\in \mathbb{T}^n\times \mathbb{R}: y < f(x,t)}$, then at a point $\hat{x} = (x,f(x))\in \partial \Omega$, the outward unit normal vector to $\partial\Omega$ is given by $$ \mathbf{n}(\hat{x}) = \frac{1}{\sqrt{1+|\nabla f(x)|^2}} \begin{pmatrix} - \nabla f(x) \ 1 \end{pmatrix}. $$
