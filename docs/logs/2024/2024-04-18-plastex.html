Notes on plastex

At the time of writing this post, plastex has the following issues:

  • If an equation has a special name with tag{abc}, then after running plastex the label of the equation gets duplicated, which making error when referring to the equation in the body of the text.

    (Temporary) Solutions: remove all special tags, just using the automatic numbering built-in in Latex.

    Also, if a label of an equation contains abc_xyz it will causes problem, try abc-xyz instead.

  • tikz does not work well, try using .png for figures (not even .pdf figures).

  • vspace{} does not work, try to remove them.

  • enumitem with labels for description enviroment does not work.

  • Cannot use .bib directly. One needs to render .bbl first and then use that
